project overview

our school was hosting an interschool arts fest called Sahodhya where schools from the entire thrissur district will participate. i was tasked with building a digital map which would show locations of each stage/event across the school campus.

the webapp helped multiple people navigate to their specific location and also helped them explore the school campus.

tech stack & tools

  • leaflet – lightweight javascript library for interactive maps
  • openstreetmap – free and open source map tiles used as the base layer
  • git – version control system to track code changes and collaborate
  • github – used for project control and team collaboration
  • github pages – static site hosting the webapp
  • html – markup language used to structure web content
  • css – stylesheet language to design the look and feel of web pages

key features

event locations

shows map markers for upcoming club or campus events, letting users quickly visualize where activities take place.

your location

displays the user's current location on the map, helping them navigate and find nearby events and facilities.

medical room finder

pinpoints the exact locations of medical rooms on campus, making it easy for users to locate help in emergencies.

essentials map

shows important amenities like washrooms and water points, so users can quickly find the facilities they need.

challenges & solutions

implementing real-time location tracking of the user

problem: needed to reliably display each user's live location on the map in real time across a wide range of devices with privacy and accuracy considerations.

solution: used the browser's geolocation api with frequent updates and integrated it into leaflet maps, plus deployed secure websockets to sync updates—making sure users always see their accurate live location and that permissions and privacy are respected.

designing responsive interface for mobile use

problem: almost all users would access the website on mobile devices during events.

solution: implemented mobile-first design with tailwind css, focusing on touch-friendly interfaces and optimized loading speeds.

results & impact

guide map helped make the art fest much more accessible to people who were not familiar with the school. it is being continously altered and updated by school students for ongoing events.
