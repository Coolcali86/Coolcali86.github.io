energy conservation

automatic class energy management system

smart room occupancy detection and automated appliance control

status: completed
recognition: aim green innovators '23 top 100
energy saved: estimated 20-30% per room

project overview

noticed how much energy was wasted in empty classrooms with lights and fans running all day. this system uses computer vision to detect human presence and automatically controls room appliances. it can distinguish between actual people and objects, reducing false positives.

the system combines multiple detection methods for reliability - computer vision as primary detection, with pir sensors as backup. it includes energy monitoring to track actual savings and can be controlled remotely via wifi.

components & hardware

main processing

  • raspberry pi 3b+ - main computer for vision processing and decision making
  • arduino uno - relay control and sensor interfacing
  • microsd card (16gb) - operating system and data logging

detection sensors

  • usb camera (logitech c270) - computer vision based person detection
  • pir motion sensor (hc-sr501) - backup motion detection
  • infrared sensors - entry/exit point monitoring

control & monitoring

  • 4-channel relay module (5v) - high-power appliance switching
  • current sensors (acs712) - real-time power consumption monitoring
  • esp8266 wifi module - remote monitoring and control
  • real-time clock (ds1307) - time-based scheduling

user interface

  • lcd display (20x4) - system status and energy consumption
  • rotary encoder - manual settings adjustment
  • status leds - system operational indicators
  • buzzer - audio alerts for system events

how it works

1

person detection

yolo object detection algorithm identifies people in the camera feed, distinguishing them from furniture and other objects with 95% accuracy.

2

occupancy tracking

person counting algorithm tracks people entering and leaving through designated entry points, maintaining accurate room occupancy count.

3

intelligent control

based on occupancy data and time schedules, the system automatically turns appliances on/off with configurable delays to prevent false triggers.

4

energy monitoring

current sensors measure real-time power consumption and log data to calculate energy savings and generate reports.

challenges & solutions

distinguishing humans from moving objects

problem: initial motion detection triggered false positives from moving chairs, curtains, and shadows.

solution: implemented yolo v3 object detection specifically trained to recognize people, combined with size and movement pattern analysis to filter false positives.

handling multiple people entering/leaving

problem: accurately counting people when multiple individuals enter or exit simultaneously through the same door.

solution: developed tracking algorithm using centroid tracking and kalman filters to maintain individual person identities across frames.

reliable relay control for high-power appliances

problem: switching high-power fans and lights required proper electrical isolation and safety measures.

solution: used optoisolated relays rated for 230v ac loads, implemented proper grounding, and added surge protection to prevent damage.

energy consumption tracking

problem: measuring actual energy savings required accurate current sensing and data logging over time.

solution: integrated acs712 current sensors with calibration routines and implemented data logging with sqlite database for long-term analysis.
