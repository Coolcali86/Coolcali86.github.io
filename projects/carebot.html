carebot.



personal care robot with face detection and vital signs monitoring

status: prototype
recognition: National Digital Fest & Global Indian Digital Fest Champion

project overview

this project came from seeing my grandmother struggle with remembering to take medications. carebot is an autonomous robot that can navigate to patients, dispense the correct medication and water, and monitor vital signs. it uses facial recognition to identify different family members and keeps track of their medication schedules.

the robot combines computer vision, autonomous navigation, and healthcare monitoring into a single platform. it's designed to be non-intrusive while ensuring patients receive their medication on time and their health is continuously monitored.

components & hardware

main computing

  • raspberry pi 4b - main computer for vision processing and decision making
  • arduino uno - sensor control and actuator management
  • microsd card (32gb) - operating system and data storage

vision & sensing

  • usb webcam - for face & body detection and navigation
  • ultrasonic sensors (hc-sr04) - obstacle detection and navigation
  • temperature sensor (ds18b20) - body temperature monitoring
  • pulse sensor - heart rate monitoring
  • gsm module - to send information to caretaker

movement & control

  • dc motors with encoders - precise wheel movement
  • motor driver (l298n) - motor speed and direction control
  • servo motors (sg90) - pill dispenser mechanism
  • custom chasis made of multiwood - robot frame

user interface

  • led indicators - system status and emergency alerts
  • push buttons - to begin reading of heartrate and temperature

how it works

1. face recognition

opencv-based face & body detection identifies family members and matches them to their medication schedules stored in a json database.

2. autonomous navigation

using the webcam, the robot navigates through the room to find the correct person for medication delivery.

3. medication dispensing

custom 3d-printed dispenser with servo-controlled mechanism releases the exact number of pills based on the prescription.

4. vital signs monitoring

temperature and pulse sensors take readings while dispensing medication, logging data for caretaker.

challenges & solutions

accurate face detection in different lighting

problem: opencv's default face detection struggled with varying indoor lighting conditions.

solution: implemented body detection as well to help with face detection.

precise pill dispensing mechanism

problem: needed to dispense exact number of pills without jamming or miscounting.

solution: designed custom handbuilt rotating system with servo-platter.

autonomous navigation in home environment

problem: navigating around furniture and people in a dynamic home environment.

solution: implemented basic slam using multiple ultrasonic sensors and created a simple occupancy grid map that updates in real-time.
